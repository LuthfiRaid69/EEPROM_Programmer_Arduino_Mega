# EEPROM_Programmer_Arduino_Mega
EEPROM Programmer for : 29LV160, 29LV320, 29LV640, Etc. internal memory SP404 OG , Juno D and any Pedal FX &amp; Synthesizer.
##
### Pin Connecting

| EEPROM Pin | Arduino Pin |
|------------|-------------|
| A0         | 11          |
| A1         | 8           |
| A2         | 62          |
| A3         | 48          |
| A4         | 63          |
| A5         | 46          |
| A6         | 64          |
| A7         | 44          |
